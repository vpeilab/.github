# VPEI Lab

**Visual-Native Predictive Embodied Intelligence Laboratory**

We are building an AI system that learns the causal structure of the world from scratch—through its body, through prediction errors, and through nothing else.

---

### The Three Iron Laws

1. **Reality Boundary:** Cognitive modules receive only pixels and actions. No coordinates, no labels, no collision events.
2. **Zero Pretraining:** Every causal learning experiment starts from random weights. No borrowed knowledge.
3. **Local Learning Only:** Synaptic updates are local. Global backpropagation is forbidden in the core system.

### How to Verify (Not Benchmark)

We do not chase SOTA on ImageNet. We chase **falsifiable predictions** about intelligence.

- **Tier 1:** Does it explore without reward?
- **Tier 2:** Does it learn causality in 5 shots?
- **Tier 3:** Does it invent tools and chain goals?
- **Tier 4:** Can it survive a blind-box causality test?

If a model passes all four tiers and the baselines fail, we call it intelligent.
